¿Cómo funciona?

A través de un entrenamiento exhaustivo de datos de millones de modelos de curriculum hemos conseguido una inteligencia artificial capaz de ver los detalles en los que se centran los empleadores. Simplemente tienes que dar un pequeño contexto de a lo que quieres dedicarte, un estilo que te gustaría y la IA se encargará de hacer el trabajo duro por ti. ¡Si no te gusta el diseño tienes hasta 3 veces de forma gratuita al día para reintentarlo!

¿Qué datos recogemos?

Tranquilo/a, tus datos están a salvo, nos comprometemos a no usar los datos ni para entrenar a nuevos modelos ni para vendersélos a terceros. Estos datos solo se guardarán para que tu dispongas de tu curriculum cuando quieras. Siempre puedes borrarlos manualmente o programar un tiempo definido para que estos se borren permanentemente al cabo de un tiempo.
